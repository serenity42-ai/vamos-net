# Padel Scoring System: A Complete Breakdown

Padel scoring is tennis scoring, point for point. The 15–30–40 game progression, the deuce, the set, the tiebreak. All identical. The only thing tennis players have to learn is one rule, the **golden point**, and that's mostly because it's the rule that decides every game on the professional Premier Padel circuit.

Here's how the whole system works.

## The basics: points, games, sets

Each game runs through this sequence:

- **0 (Love)** — no points scored
- **15** — first point won
- **30** — second point won
- **40** — third point won
- **Game** — fourth point won (if leading)

A pair needs four points to take the game, with at least a one-point lead after both reach 40. When the score is 40–40, the game is at **deuce**. How deuce gets resolved depends on whether the match uses the advantage system or the golden point rule, both covered below.

### Sets

A set goes to the first pair to win **six games**, with a minimum lead of two. So a set can end 6–0, 6–1, 6–2, 6–3, or 6–4 cleanly.

If the set reaches 5–5, the next game makes it either 6–5 or 5–6. Pressure on the trailing pair. If they level at 6–6, a **tiebreak** decides the set.

### Matches

A standard padel match is **best of three sets**. The first pair to two sets takes the match. There is no best-of-five format in padel, even in professional finals. Most matches finish in 60–120 minutes, which is one reason the sport translates so well to live broadcast schedules.

## Deuce and the advantage system

Under traditional scoring, when the game is at 40–40 (deuce), the next-point winner gets the **advantage**. Win the following point, take the game. Lose it, back to deuce. A game at deuce can theoretically swing back and forth forever until one pair manages to win two consecutive points from there.

When the game reaches deuce, the receiving pair chooses which side of the court (left or right) to receive on. Most pairs put their stronger returner on the left (the backhand side for right-handed players), because that's the side from which the highest-pressure return needs to come back.

## The golden point rule

The most distinctive feature of professional padel scoring. Introduced to compress matches and add tension, golden point replaces the advantage system at deuce.

How it works:

1. At 40–40, only **one deciding point** is played.
2. The **receiving pair** chooses which side of the court to receive on.
3. Whoever wins that single point wins the game.

No advantage. No need to win two in a row. One point decides everything. That's the source of most of the dramatic moments in pro padel.

### Where it's used

Golden point is the standard on the **Premier Padel** circuit, and it was the standard on the World Padel Tour before that. Most professional tournaments now use it.

Plenty of amateur leagues, club competitions, and recreational matches still use traditional advantage. Worth checking with your opponents before the match starts which system you're playing under.

### Strategic implications

Golden point fundamentally changes deuce dynamics:

- **Serving pair:** added pressure. A single lapse on a single point costs the game, regardless of how cleanly they served the rest of the way.
- **Receiving pair:** the side choice is genuinely tactical. Pairs put their strongest returner on the chosen side and load the answer there.
- **Match tempo:** games can't drag through endless deuces. Match length becomes more predictable, which matters for broadcast and for booking-window planning.

## The tiebreak

When a set reaches 6–6, a tiebreak decides it. Padel uses the standard tennis tiebreak:

1. The pair due to serve next starts with **one serve** from the right (deuce) side.
2. After that, each pair serves **two consecutive points**, alternating between right and left sides.
3. Players switch ends every **six points**.
4. The first pair to **seven points** with a lead of at least two wins the tiebreak and the set.
5. If the tiebreak reaches 6–6, play continues until one pair leads by two.

Tiebreak scoring is numerical (1, 2, 3, 4...) rather than 15–30–40. Final scores like 7–5 or 9–7 are normal.

In some tournament formats, if the match reaches one set all and the deciding set also goes to 6–6, a **super tiebreak** (first to 10, win by two) replaces the standard tiebreak. Not universal; depends on the competition.

## Service rules and how they affect scoring

Padel service is underarm and tightly regulated:

- Strike the ball at or below waist height
- Serve from behind the service line, into the diagonal service box
- The ball must bounce once in the service box before the returner can play it
- If the ball clips the net and lands in the correct box, it's a **let** and the serve is replayed
- If the ball clips the net and then touches the side or back wall before bouncing in the service box, it's a **fault**
- Two attempts per point (first and second serve)
- A **double fault** loses the point

The pair alternates servers each game; within a game, one server stays in. The order within a pair is set at the start of each set and is fixed for that set.

## Match formats in professional padel

### Premier Padel tour

- Best of three sets
- Golden point at deuce
- Standard tiebreak at 6–6 in each set
- No final-set super tiebreak (standard tiebreak applies)

### Club and amateur matches

Recreational padel runs on variations:

- **Best of three sets** with advantage scoring (no golden point)
- **Single set to 9 games** with tiebreak at 8–8 (common for booking-window time pressure)
- **Timed matches**, where whoever leads when the clock runs out wins
- **Round-robin formats** with shortened sets (first to 4 or 6 games)

These variations exist mostly to fit matches inside standard 90-minute court bookings.

## Scoring at a glance

**Points in a game:** 0, 15, 30, 40, Game

**Games in a set:** First to 6 games, win by 2. Tiebreak at 6–6.

**Sets in a match:** Best of 3. First to 2 sets takes it.

**At deuce (40–40):**
- Golden point: one deciding point, receiving pair chooses side
- Advantage: must win two consecutive points from deuce

**Tiebreak:** First to 7 points, win by 2. Numerical scoring.

## Common scoring scenarios

### A straightforward game
Serving pair wins four straight: 15–0, 30–0, 40–0, Game. Called a "love game."

### Deuce with golden point
Score reaches 40–40. Receiving pair picks their side. One point. Winner takes the game.

### Tiebreak set
Set reaches 6–6. Tiebreak. Pair A wins it 7–4. Pair A wins the set 7–6.

### Deciding set
Match tied at one set each. The third set runs by the same rules. If it reaches 6–6, standard tiebreak (or super tiebreak, depending on tournament rules) decides the match.

## How padel scoring differs from tennis

| Feature | Padel | Tennis |
|---|---|---|
| Golden point | Standard in pro play | Not used (advantage standard) |
| Match length | Best of 3 sets | Best of 3 or 5 |
| Serve style | Underhand only | Overhand |
| Deuce side choice | Receiving pair picks | No side choice |
| Let on wall contact | Fault if ball touches wall after net | Not applicable |

## Tips for keeping score

If you're scoring a match yourself:

- **Call the server's score first.** If the serving pair has 30 and the receiving pair has 15, the score is "30–15."
- **Track games clearly.** Tally on paper, or use the on-court scoreboard.
- **Agree on deuce rules before the match.** Golden point or advantage. One question, settled in five seconds.
- **Remember service order.** Each player serves a full game in turn. Order within a pair stays fixed for the set but can change between sets.

That's the whole system. Tennis players will be running matches by the second set. Newcomers usually pick it up by the third. Either way, the only thing that matters once the score is clear is the rally.
